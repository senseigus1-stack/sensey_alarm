Будильник Sensey
Описание проекта
Alarm Sensey — это многофункциональное приложение-будильник с графическим интерфейсом на Python. Программа позволяет устанавливать будильники, настраивать звуковые оповещения, выбирать устройства вывода звука и управлять громкостью.

Особенности
Графический интерфейс на основе Tkinter

Настройка времени будильника через удобный интерфейс

Выбор устройства вывода звука

Регулировка громкости в реальном времени

Функция Snooze (отложить будильник)

Сохранение настроек в конфигурационный файл

Логирование действий в файл

Установка
Клонируйте репозиторий:

bash
git clone https://github.com/your-username/alarm-clock.git
cd alarm-clock
Создайте виртуальное окружение:

bash
python -m venv venv
source venv/Scripts/activate  # Для Windows используйте venv\Scripts\activate
Установите зависимости:

bash
pip install -r requirements.txt
Требования
Python 3.8+

Библиотеки:

tkinter (входит в стандартную поставку Python)

pygame

sounddevice

numpy (требуется для sounddevice)

pyyaml

python-dateutil

Использование
Запустите приложение:

bash
python alarm_clock.py
В интерфейсе установите:

Время будильника

Устройство вывода звука

Уровень громкости

Нажмите кнопку “Установить будильник”

Структура проекта
alarm_clock/
├── alarm_clock.py
├── config.json
├── Свиридов Время Вперед.wav
└── requirements.txt
Конфигурация
Основные настройки хранятся в файле config.json:

json
{
    "last_time": {
        "hour": "00",
        "minute": "00"
    },
    "device": null,
    "sound_file": "Свиридов Время Вперед.wav",
    "volume": 0.5
}
Логирование
Все действия приложения записываются в файл alarm_clock.log с указанием времени, уровня важности и сообщения.
