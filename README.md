# Alarm Sensey

Простой будильник с настраиваемыми звуком, громкостью и аудиоустройством.


## Функции

- Установка времени будильника.
- Выбор аудиоустройства вывода.
- Регулировка громкости.
- Функция Snooze (отложить на 5 минут).
- Сохранение настроек в `config.json`.
- Логирование действий в `alarm_clock.log`.

## Установка

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt

2. Запустите
    ```
    python src/main.py

3. Настройка
    Звук будильника можно изменить, отредактировав config.json (поле sound_file).

    Громкость сохраняется в config.json (поле volume).
